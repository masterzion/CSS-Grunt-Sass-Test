Folders
=======
./                     : development enviroment configurated to "http://localhost/jairo_crytest"
images/                : Imagens files
js/home.js             : uncompressed script
production/            : compressed files
sass/                  : sass - build with http://compass-style.org/
stylesheets/screen.css : final CSS (uncompressed)
config.rb              : compass configuration file
index.html             : development index file

BONUS
=====
Gruntfile.js           : Grunt script (to build production/ )
package.json           : Grunt configuration file
- stylesheets/fonts    : Fonts (I have not received it in the mail)
- Fade effect
- Sprites
- Using Compass



COMMANDS
========
-- install compass -- 
$sudo gem install compass

-- build css -- 
$compass compile

-- install grunt-- 
$npm install

-- compress and copy files-- 
$grunt
